Explorando la robótica desde el mundo de las ciencias

El programa apoya y aporta las herramientas necesarios para que todos los niños, niñas y jovenes aprendan de ciencias a traves del juego y la curiosidad.

Ponemos al alcance de los lectores de este sitio, todos los programas utilizados y los vectores creados durente el trancurso del ciclo de aprendisaje de la maquina de corte por CO2.

¿Que aprendemos Desde la programación?

Desarrollo web y Apps a medida de nustras necesidades

  • Aplicaciones Web

    Desarrollo web usando HTML, CSS3 Y Java Script.

  • Aplicaciones con Arduino

    Uso de Arduino como herramienta de aprendizaje en Hardware y Software.

  • Raspberry Pi

    Uso de Raspberry Pi, como herramienta de aprendizaje e internet de las cosas.

  • Juegos en Scratch

    Aprender la lógica de programación usando Scratch para la creación de videojuegos.

  • Lógica de programación

    Buscamos herramientas que nos ayuden a llevar a cabo nuestros proyectos.
